# ePayco Recaudo Landing

Landing page estática para la página de recaudo en línea de ePayco.

## Estructura

- `index.html`: redirección a la landing.
- `recaudo-en-linea/index.html`: página principal con SEO y FAQ schema.
- `recaudo-en-linea/styles.css`: diseño responsive.
- `recaudo-en-linea/script.js`: interacción, acordeones y animaciones GSAP con ScrollTrigger.

## Vista local

Abre `index.html` en el navegador o sirve la carpeta con cualquier servidor estático.
